## Operations
Each operation has the general format $\left\langle F, i, j, k, K \right\rangle$.

### The $F$ field

| $F$        | Operation           | Description              |
|------------|---------------------|--------------------------|
| 0xb3       | **agen**            | Address generatin        |
| 0xb2       | **cmp**             | Compare with a register  |
| 0x25       | **rdw**             | Read data word           |
